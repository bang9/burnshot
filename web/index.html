🔥 Burnshot

Proof-of-burn for AI coding sessions

Snap your token stats like Nike Run Club snaps your miles.

How it works

  1. Run burnshot in your terminal — collects token usage from Claude Code & Codex CLI
  2. Scan the QR code with your phone
  3. Snap a photo — your token stats overlay on the camera viewfinder
  4. Share — save or post your proof-of-burn

Install

curl -fsSL https://raw.githubusercontent.com/bang9/burnshot/main/install.sh | bash
